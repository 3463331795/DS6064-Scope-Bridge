# Changelog

## 0.1.0 - 2026-07-11

- Added safe JSON CLI for RIGOL DS6064 / DS6000 USB-TMC access.
- Added PyVISA/NI-VISA link checks, resource probing, and watchdog-protected hardware commands.
- Added built-in scalar measurements for frequency, period, duty cycle, and Vpp.
- Added single-channel and multi-channel waveform capture to CSV, PNG, and manifest files under `outputs/`.
- Added `snapshot` as the preferred AI handoff command for multi-channel evidence packages.
- Added Codex skill files for natural-language oscilloscope workflows.
- Added offline parser, lock, manifest, and waveform analysis tests.
